GitHub Visualisation and Recommendation

Lang-Visual Geo-Visual Recommendation

Work In Progress!!!!!

  • Our Project aims at visualising the use of programming languages used in software projects on GitHub

  • We have used the GitHub Timeline dataset of the year 2013.

  • We aim to portray "Who Knows What on GitHub" by using the Chord Diagram D3-Graphical interface.

  • We made use of the repo_owner and repo_lang fields to analyse the usage of combinations of two languages together and represented it in the form of a chord diagram.

  • We chose to concentrate on the top 15 languages being used on software projects on GitHub.

Tech Stack Used

  • Play 2.0 Framework is the backbone environment of our Project.

  • BigQuery : Google's datastore that is housing the github dataset.

  • D3.js : For the Languages visualisation

  • Google Charts : For the Geographical Distribution visualisation.
